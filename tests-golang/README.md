# Primeiros Passos Golang
- Para cima das versões mais novas do Go Modules **São Obrigatórios.**
- O que são Modulos? 
	- Unidade lógica de Código
	- Namespace
	- Contexto de compilação
- Para escrever testes em go basta ele conter o sufixo **_test.go**
- Além do mais que a função de teste necessáriamente precisa começar com a palavra reservada **Test**
